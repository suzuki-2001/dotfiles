  aerc - keybindings
  ------------------

  Navigate                                  Search
    j / k        move down / up               /            search messages
    Ctrl-d / u   half-page down / up          n / N        next / prev result
    g / G        first / last message        \            filter the list
    Enter        open selected mail          Esc          clear search/filter
    q            close view, or quit

  Folders                                    Accounts
    J / K        next / prev folder          Tab / S-Tab  switch account
    H / L        fold / unfold tree
    c <name>     jump to folder

  Reply & write                              Compose / review
    rr / Rr      reply all / to sender       Tab / C-j    next field
    rq / Rq      reply, quoting text         S-Tab / C-k  previous field
    f            forward message             C-o          complete address
    C            compose new mail            y / n        send / discard
                                             e / a        edit / attach

  Organize                                   Commands
    d / D        delete: confirm / now       :cf          change folder
    A            archive message             :next-tab    switch account
    v / V        mark / range-select         :connect     reconnect account
                                             :abort       cancel a compose
                                             :help        open the manual

  To cancel a compose: leave the editor, then run :abort
